class ConditionalIgnoreRule(realDeviceEvaluator: RealDeviceEvaluator = DefaultRealDeviceEvaluator()) : MethodRule

Rule to be used to ignore certain tests under certain conditions based on test methods annotated with ConditionalIgnore, RequiresEmulator or RequiresRealDevice.

open override fun apply(base: Statement?, method: FrameworkMethod?, target: Any?): Statement?

Modifies org.junit.runners.model.Statement to be executed by a org.junit.Test method. This method is used to detect whether a given test method is annotated with ConditionalIgnore, RequiresEmulator or RequiresRealDevice, and if so a new org.junit.runners.model.Statement is returned that will ignore the execution of the test.
